# Security Policy

## Reporting a vulnerability
Please open a **private security advisory** in GitHub (Security → Advisories) or contact the maintainers at: **[add-security-email@your-org]**.

## Do
- Provide steps to reproduce and affected versions.
- Include impact analysis and proposed mitigations if available.

## Don’t
- Don’t open public issues for vulnerabilities.
- Don’t share exploit details outside the advisory thread.

We aim to acknowledge within 2 business days and coordinate a safe disclosure timeline.
